# ClearLearn — 综合学习智能体

一款集成多领域学习资源与 AI 智能答疑的 Web 应用，聚合英语词汇、驾考理论、历史知识四大板块，每个板块配备独立的 DeepSeek AI 对话助手，为用户提供一站式智能学习体验。
